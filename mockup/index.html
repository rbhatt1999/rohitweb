// rohitweb / redesign / preview · v2 (3d + scroll motion)

Terminal/dev redesign — perspective-projected 3D, scroll-driven

Real perspective 3D wireframe terrain, hero arrow, and project meshes — all projected through a virtual camera that dollies forward as you scroll. Sections enter with 3D rotateX + translateZ tied to scroll. Each project card carries a rotating 3D mesh (icosahedron for Sadhak AI, house for Stay-a-While). The contact orbit is a real 3D ring. Production build will swap canvas-2D for React Three Fiber + GSAP ScrollTrigger; this preview is faithful to the motion language.
