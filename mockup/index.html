// rohitweb / redesign / preview

Terminal/dev redesign — static visual mockup

Side-by-side preview of the desktop (1280×800) and mobile (375×812) layouts. The actual build will use React Three Fiber + drei + postprocessing for the 3D wireframe background and per-project mini canvases, GSAP + ScrollTrigger for scroll-scrubbed sequences, Lenis for smooth scroll, and keep Framer Motion for component transitions. This HTML uses canvas 2D + CSS to approximate the look — final version will be considerably richer.
